# Storage Notes for KCNA Preparation

This folder contains concise notes on Kubernetes storage concepts.

## Topics covered
- Volumes in Kubernetes
- Persistent Volume and Persistent Volume Claim
- Storage Class
- Container Storage Interface
- Storage Drivers
- Volume Drivers

## Quick revision checklist
- Understand the difference between volume, PV, PVC, and StorageClass.
- Know the access modes: ReadWriteOnce, ReadOnlyMany, ReadWriteMany.
- Know the difference between static and dynamic provisioning.
- Remember that CSI provides a standard interface for storage plugins.
